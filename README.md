# SynapsePDM-Enclosure
This is the CAD data for the SynapsePDM enclosure.

## Connectors Used

For the main power into the controller, the cutout accepts an Amphenol SurLok SLPRBBPSR which mates to a SLPPB50BSR0 on the cable side. See https://www.amphenol-industrial.de/en/products/xev-connectors/surlok for more info.

For the USB connection, the cutout accepts an E-Weichat EW-LP16-TYPE-C which mates to an LP16 Type-C male plug or a regular USB type-C connector. See https://www.e-weichat.com/Products/info_pcid_278_itemid_947.html for more info.
